    PROJECTS

    KUMON ENROLLMENT PREDICTION SYSTEM

    This system was built to predict whether the child studying in Kumon will be able to continue enrolling for the next school year. The attributes used was the child's demographic profile, his/her evaluation data and other Kumon data. After gathering the data, our team performed a data cleaning process using Microsoft Excel and SPSS software. After that, we performed an analysis and evaluation to examine which of the chosen existing predictive algorithms were able to get the required accuracy, by testing the attributes using the WEKA software.

    After the evaluation was finished, the predictive result of the chosen algorithm was being transferred into a JavaScript code which is being applied during the development of the website. The website then was developed using and HTML and CSS, and implemented a dashboard feature, which was created using Microsoft Power BI to show the summarized context of Kumon's enrollment season during the year of our development (2023).

    TECH-VOC AND SHS GRADUATES DASHBOARD

    This applies the descriptive analysis of Metro Manila's educational sector using data visualization, however to be precise, the dashboard was intended to output the comparison between the number of Technical-Vocational and Senior High School graduates.

    This output will aim to identify the potential skill gaps and other differences between the Tech-Voc and SHS sectors. It functions as a metric for identifying areas within the Metro Manila that needs improvement.

    THE PET HAVEN WEBSITE

    This website was developed using HTML, CSS, JavaScript and PHP. This website is mainly for the users who would like to adopt cats and dogs. The website consists of adopt features where they can fill up a form, perform an evaluation and set up a schedule for the adoption day. The website also offers a feature of showing lists of lost & found animals, where the data could be gathered from different animal lost & found centers within the municipality of Malolos.

    Finally, Pet Haven includes rehoming feature, where users can submit their pets for adoption just by filling up their forms. The website also accepts all forms of donations.

    AQUA WEBSITE

    Aqua Website mainly shows measurement on the harvests of aquaculture farms from Paombong, Bulakan, Malolos, and Hagonoy Provinces. The website integrates a map feature where users can easily locate and track different aquaculture farms.

    It has a dashboard feature containing data of harvested species every six months, the data was being gathered from the farm workers across the municipalities. Data cleaning steps and descriptive analysis were applied during the development process. The website were developed using ReactJS framework, and firebase was being implemented as the database.
